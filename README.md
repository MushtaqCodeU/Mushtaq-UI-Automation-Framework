# Mushtaq-UI-Automation-Framework

This project is a UI Automation Framework built using Selenium WebDriver, TestNG, and the Page Object Model (POM) design pattern to ensure maintainability, readability, and scalability of automated test scripts.

## Features
- Page Object Model (POM)
- Modular and maintainable architecture
- Easy to extend
- Maven based

## Tech Stack
- Java
- Selenium
- TestNG
- Maven

## Setup

Clone:
git clone https://github.com/MushtaqCodeU/Mushtaq-UI-Automation-Framework.git

Install:
mvn clean install

Run:
Open testRunner/testng.xml
